Les 3 Axiomes d'une Norme

Définition. N : E → ℝ⁺ est une norme si :
① N(x) ≥ 0 et N(x)=0 ⟺ x=0

Séparation — seul 0 a norme nulle. La sphère ne passe jamais par l'origine.

② N(λx) = |λ| · N(x) ∀λ∈ℝ

Homogénéité — la surface est centrée et symétrique. Changer p déforme la sphère.

③ N(x+y) ≤ N(x) + N(y)

Inégalité triangulaire — la surface est convexe. Pour p≥1 seulement !

Observer : Pour p < 1 la surface n'est plus convexe → ce n'est plus une norme.

Norme Lp sur ℝ³

‖x‖ₚ = (|x₁|ᵖ + |x₂|ᵖ + |x₃|ᵖ)^(1/p)

La boule unité B(0,1) = {x : ‖x‖ₚ ≤ 1} change de forme selon p :

  • p=1 → Octaèdre régulier (8 faces triangulaires)
  • p=2 → Sphère euclidienne parfaite
  • p→∞ → Cube [−1,1]³
  • 1≤p≤∞ → Corps convexe (norme valide)
Inégalité de Hölder : Pour 1/p + 1/q = 1 :
|⟨x,y⟩| ≤ ‖x‖ₚ · ‖y‖q
Cas limite p=0.5 : la forme n'est plus convexe → pas une norme !

Boules dans (E, ‖·‖)

B(a,r) = {x : ‖x−a‖ < r} ouverte
B̄(a,r) = {x : ‖x−a‖ ≤ r} fermée
S(a,r) = {x : ‖x−a‖ = r} sphère

La boule ouverte (fil rouge) n'inclut pas sa frontière.

La boule fermée (semi-transparent) inclut la sphère frontière.

  • ‖·‖₂ → sphère ronde
  • ‖·‖₁ → octaèdre
  • ‖·‖∞ → cube
Propriété : Les boules ouvertes forment une base de la topologie induite par la norme.

Suites de Cauchy

Définition. (xₙ) est de Cauchy si :
∀ε>0, ∃N : ∀p,q≥N, ‖xₚ−xq‖ < ε

La spirale se resserre vers un point. Les points dorés sont les termes. La boule bleue représente ε-voisinage.

∀p,q ≥ N(ε) : ‖xₚ − xq‖ < ε
  • Toute suite convergente est de Cauchy
  • La réciproque ⟺ espace complet
  • Mode "non-complète" : la limite sort de l'espace !
Exemple : Dans ℚ, la suite des approximations de π est de Cauchy mais π∉ℚ.

Espace de Banach

Définition centrale. (E,‖·‖) est un espace de Banach si toute suite de Cauchy converge dans E.
Exemples de Banach :
• (ℝⁿ, ‖·‖ₚ) pour tout p≥1
• (C([a,b]), ‖·‖∞)
• ℓᵖ(ℕ) pour tout p≥1
• Lᵖ(Ω) pour tout p≥1
Non-Banach : C([0,1]) avec ‖f‖₁=∫|f| n'est pas complet.
Son complété est L¹([0,1]).

Dans la simulation : la spirale rouge converge vers le point bleu (Banach) ou s'échappe de la sphère (non-complet).

VI Topologie des Espaces Normés
INTÉRIEUR · ADHÉRENCE · FRONTIÈRE (3D)
⟳ drag

Topologie Induite par la Norme

Int(A) = ⋃{B(x,r) ⊂ A}
Ā = {x : ∀r>0, B(x,r)∩A ≠ ∅}
∂A = Ā \ Int(A)
  • Intérieur : points avec une boule entière dans A
  • Adhérence : points limites de suites dans A
  • Frontière : ni intérieur ni extérieur
  • Dense : Ā = E entier
Théorème de Baire : Tout espace de Banach est un espace de Baire : intersection d'ouverts denses reste dense.

Norme d'Opérateur

‖T‖ = sup{ ‖Tx‖ : ‖x‖≤1 }

L'image de la boule unité par T est un ellipsoïde. Les demi-axes σ₁≥σ₂≥σ₃ sont les valeurs singulières de T.

‖T‖ = σ₁ (plus grande valeur singulière)
  • Si T est une isométrie : σ₁=σ₂=σ₃=1 (sphère)
  • T non borné ↔ σ₁=+∞
  • T compact ↔ σₙ→0
Théorème. L(E,F) est un espace de Banach si F est de Banach, muni de la norme d'opérateur.
