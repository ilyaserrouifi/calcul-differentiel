I Espaces Vectoriels Normés
Définition. Une norme sur un espace vectoriel E (sur ℝ ou ℂ) est une application N : E → ℝ⁺ vérifiant trois axiomes fondamentaux.
Visualisation géométrique des axiomes

Les 3 Axiomes d'une Norme

Une application N : E → ℝ⁺ est une norme si et seulement si :

① N(x) ≥ 0 et N(x) = 0 ⟺ x = 0

Séparation : Seul le vecteur nul a une norme égale à zéro. La norme mesure la "taille" non nulle de tout vecteur non nul.

② N(λx) = |λ| · N(x) ∀λ ∈ ℝ, ∀x ∈ E

Homogénéité : Multiplier un vecteur par un scalaire λ multiplie sa norme par |λ|. L'échelle est absolue.

③ N(x+y) ≤ N(x) + N(y)

Inégalité triangulaire : Le chemin direct est toujours plus court (ou égal) que le détour par un point intermédiaire.

Normes Classiques sur ℝⁿ

Pour x = (x₁, x₂, ..., xₙ) ∈ ℝⁿ, les normes les plus utilisées sont :

‖x‖₁ = |x₁| + |x₂| + ... + |xₙ|

Norme de la somme absolue, donne un losange en dimension 2.

‖x‖₂ = √(x₁² + x₂² + ... + xₙ²)

Norme euclidienne, donne un disque parfait.

‖x‖∞ = max(|x₁|, |x₂|, ..., |xₙ|)

Norme du sup, donne un carré en dimension 2.

‖x‖ₚ = (|x₁|ᵖ + ... + |xₙ|ᵖ)^(1/p)

Glissez le curseur pour voir la boule évoluer continûment de ‖·‖₁ vers ‖·‖∞ !

II Normes Équivalentes
Définition. Deux normes N₁ et N₂ sur E sont dites équivalentes s'il existe α, β > 0 tels que : α·N₁(x) ≤ N₂(x) ≤ β·N₁(x) pour tout x ∈ E.
Sandwichage entre deux normes

Équivalence de Normes

Deux normes sont équivalentes si leurs boules unitaires sont "emboîtées" : l'une peut être sandwichée entre deux homothétiques de l'autre.

α · N₁(x) ≤ N₂(x) ≤ β · N₁(x)

Géométriquement : la boule unité de N₂ (bleue) est contenue dans la boule de rayon β pour N₁ (rouge grand) et contient la boule de rayon α pour N₁ (rouge petit).

Théorème. En dimension finie, toutes les normes sont équivalentes. Ce résultat remarquable est faux en dimension infinie !

Sur ℝⁿ, on a toujours : ‖x‖∞ ≤ ‖x‖₂ ≤ ‖x‖₁ ≤ √n · ‖x‖₂ ≤ n · ‖x‖∞

III Boules Ouvertes et Fermées
Boules ouvertes et fermées (cliquer pour placer un point)

Boules dans un Espace Normé

Dans (E, ‖·‖), on définit deux types de boules centrées en a ∈ E de rayon r > 0 :

B(a, r) = {x ∈ E : ‖x − a‖ < r} (ouverte)
B̄(a, r) = {x ∈ E : ‖x − a‖ ≤ r} (fermée)

La sphère de centre a et rayon r est :

S(a, r) = {x ∈ E : ‖x − a‖ = r}
  • La boule ouverte exclut la sphère frontière
  • La boule fermée inclut la sphère frontière
  • La forme dépend de la norme choisie !
  • Toute boule ouverte est un ouvert ; toute boule fermée est un fermé
IV Suites de Cauchy
Définition. Une suite (xₙ) dans (E, ‖·‖) est de Cauchy si : ∀ε > 0, ∃N ∈ ℕ tel que ∀p,q ≥ N : ‖xₚ − xq‖ < ε.
Suite de Cauchy qui converge vs suite qui ne converge pas

Suites de Cauchy et Convergence

Une suite de Cauchy est une suite dont les termes se resserrent mutuellement, sans nécessairement converger vers un point de l'espace !

Théorème. Toute suite convergente est de Cauchy. La réciproque est vraie si et seulement si l'espace est complet.

Exemple clé : Dans ℚ (rationnels), la suite 3, 3.1, 3.14, 3.141, 3.1415, ... est de Cauchy mais sa limite π ∉ ℚ. L'espace ℚ n'est pas complet !

‖xₚ − xq‖ < ε pour tous p, q ≥ N(ε)

Plus N est grand, plus ε peut être petit : les termes deviennent arbitrairement proches entre eux.

V Complétude — Espaces de Banach
Définition fondamentale. Un espace vectoriel normé (E, ‖·‖) est un espace de Banach si toute suite de Cauchy dans E converge vers un élément de E. (E est complet pour la distance induite par la norme.)
Espace complet vs non-complet
Mode : Espace complet (Banach)

Espaces de Banach

La complétude est la propriété qui garantit qu'aucune suite "convergente en esprit" ne peut s'échapper de l'espace.

Exemples d'espaces de Banach :
• (ℝⁿ, ‖·‖ₚ) pour tout p ≥ 1
• C([a,b]) avec ‖f‖∞ = sup|f(t)|
• ℓᵖ(ℕ) = {(xₙ) : Σ|xₙ|ᵖ < ∞}
• L²(Ω) (espace de Hilbert)
Contre-exemples (non de Banach) :
• (ℚ, |·|) — rationnels avec valeur absolue
• C([0,1]) avec ‖f‖₁ = ∫₀¹|f(t)|dt
• Polynômes P([0,1]) avec ‖·‖∞
Théorème. (ℝⁿ, ‖·‖) est un espace de Banach pour toute norme ‖·‖.
Complété d'un espace : ajouter les points manquants

Complété d'un Espace Normé

Tout espace normé (E, ‖·‖) peut être "complété" : on peut construire un espace de Banach Ê qui contient E comme sous-espace dense.

E ⊂dense Ê (Ê est de Banach)

Construction : les éléments de Ê sont des classes d'équivalence de suites de Cauchy de E.

  • ℚ se complète en ℝ
  • Les polynômes P([0,1]) se complètent en C([0,1])
  • C([0,1]) avec ‖·‖₂ se complète en L²([0,1])
Unicité : Le complété est unique à isomorphisme isométrique près.
VI Topologie des Espaces Normés
Introduction. La norme induit une topologie sur E via la distance d(x,y) = ‖x − y‖. On peut alors parler d'ouverts, fermés, compacts, connexes...
Ouverts et fermés — cliquer pour tester un point

Ouverts et Fermés

Un ensemble U ⊂ E est ouvert si tout point de U est un point intérieur :

∀x ∈ U, ∃r > 0 : B(x,r) ⊂ U

Un ensemble F ⊂ E est fermé s'il contient toutes ses limites :

F fermé ⟺ E \ F ouvert
  • ∅ et E sont à la fois ouverts et fermés
  • Union quelconque d'ouverts est ouverte
  • Intersection finie d'ouverts est ouverte
  • Intersection quelconque de fermés est fermée
  • Union finie de fermés est fermée
Dense, intérieur, adhérence

Intérieur, Adhérence, Dense

Pour A ⊂ E, on définit :

Int(A) = plus grand ouvert ⊂ A
Ā = plus petit fermé ⊃ A
∂A = Ā \ Int(A) (frontière)

A est dense dans E si Ā = E, c'est-à-dire si tout ouvert non vide de E rencontre A.

  • ℚ est dense dans ℝ
  • Les polynômes sont denses dans C([0,1]) (Weierstrass)
  • Int(B̄(a,r)) = B(a,r) dans un espace normé
VII Applications Linéaires Continues
Définition. Soit f : (E, ‖·‖_E) → (F, ‖·‖_F) linéaire. f est continue si et seulement si elle est bornée : ∃M > 0, ‖f(x)‖_F ≤ M · ‖x‖_E pour tout x ∈ E.
Application linéaire continue : image de la boule unité

Norme d'Opérateur

Pour T : E → F linéaire continue, on définit la norme d'opérateur :

‖T‖ = sup{‖T(x)‖_F : ‖x‖_E ≤ 1}
‖T‖ = sup{‖T(x)‖_F / ‖x‖_E : x ≠ 0}

C'est le plus petit M tel que ‖T(x)‖ ≤ M·‖x‖ pour tout x.

Théorème. L(E, F) = {applications linéaires continues E → F} est un espace de Banach pour la norme d'opérateur, si F est de Banach.

L'image de la boule unité de E par T est un ellipsoïde (en dim. finie).

VIII Exemples Classiques d'Espaces de Banach
Espace ℓᵖ : suites sommables

Espaces ℓᵖ(ℕ)

Les espaces de suites à puissance p-ième sommable :

ℓᵖ = {(xₙ)ₙ≥₀ ⊂ ℝ : Σₙ |xₙ|ᵖ < ∞}
‖x‖_ℓᵖ = (Σₙ |xₙ|ᵖ)^(1/p)

Pour p = ∞ :

ℓ∞ = {(xₙ) bornée}, ‖x‖_ℓ∞ = sup_n |xₙ|
  • ℓ¹ ⊂ ℓ² ⊂ ℓ∞ (inclusions strictes)
  • Chacun est un espace de Banach
  • ℓ² est un espace de Hilbert (produit scalaire)
  • Ces espaces sont séparables pour p < ∞
C([a,b]) avec ‖f‖∞ = sup|f|

Espace C([a,b])

L'espace des fonctions continues sur [a,b] muni de la norme du supremum :

‖f‖∞ = max_{t ∈ [a,b]} |f(t)|

C'est un espace de Banach fondamental en analyse. La convergence pour ‖·‖∞ est exactement la convergence uniforme.

Théorème de Weierstrass. Les polynômes sont denses dans C([a,b]). Toute fonction continue peut être approchée uniformément par des polynômes.
Attention ! C([0,1]) avec ‖f‖₁ = ∫₀¹|f| n'est PAS de Banach. Son complété est L¹([0,1]).
