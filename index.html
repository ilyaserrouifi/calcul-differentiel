I. Espaces Vectoriels Normes

1. Definition

Une norme sur le K-espace vectoriel E est une application N: E -> R+ verifiant:

i) N(x) = 0 <=> x = 0_E (Separation)
ii) N(lambda*x) = |lambda| * N(x) (Homogeneite)
iii) N(x + y) <= N(x) + N(y) (Inegalite triangulaire)

Visualisation: Les 3 normes usuelles sur R2

2. Exemples de Normes

NormeFormuleForme geometrique
||.||_1sum |x_i|Losange (diamant)
||.||_2sqrt(sum x_i^2)Cercle
||.||_p(sum |x_i|^p)^(1/p)Entre losange et carre
||.||_infmax |x_i|Carre

Contre-exemple important

Sur E = C([0,1], R), l'application N(f) = |f(0)| n'est PAS une norme car elle ne verifie pas la separation: f(x) = x verifie N(f) = 0 mais f != 0.

II. Topologie des Espaces Normes

1. Boules et Ouverts

Boule ouverte: B(x, r) = {y in E : ||y - x|| < r}
Boule fermee: B'(x, r) = {y in E : ||y - x|| <= r}

Visualisation: Boule ouverte vs fermee

Proposition: Toute boule ouverte est un ouvert

Preuve: Soit B = B(x,r) avec r > 0. Soit y in B, on pose r' = r - ||x-y|| > 0.

Alors B(y, r') est contenue dans B, donc B est un ouvert.

III. Convergence et Continuite

1. Suite convergente

(x_n) tend vers l si: forall epsilon > 0, exists N, forall n >= N: ||x_n - l|| < epsilon

2. Application continue

f est continue en x_0 si: forall epsilon > 0, exists eta > 0:
||x - x_0|| < eta => ||f(x) - f(x_0)|| < epsilon

Animation: Suite convergente

IV. Espaces Complets (Banach)

1. Suite de Cauchy

(x_n) est de Cauchy si: forall epsilon > 0, exists N, forall p,q >= N:
||x_p - x_q|| <= epsilon

2. Espace de Banach

Definition

Un espace norme (E, ||.||) est un espace de Banach (ou e.v.n. complet) si toute suite de Cauchy est convergente.

Exercice resolu: R^m avec ||.||_1 est un Banach

Solution: Soit (x_n) une suite de Cauchy dans R^m.

Alors pour chaque coordonnee i, (x_n^i) est de Cauchy dans R.

Comme R est complet, x_n^i -> l^i, donc x_n -> l dans R^m.

Exercice: Polynomes ne forment PAS un Banach

Soit P_n = sum_{k=0}^n x^k/k!. C'est une suite de Cauchy qui converge vers e^x, qui n'est pas un polynome!

Chapitre 2: Differentielle d'une Application

I. Definition

Soient (E, ||.||_E) et (F, ||.||_F) deux espaces vectoriels normes, U un ouvert de E.

On dit que f: U -> F est differentiable en x de U s'il existe L in L_c(E,F) telle que:

lim_{||h||->0} ||f(x+h) - f(x) - L(h)||_F / ||h||_E = 0
f(x+h) = f(x) + L(h) + o(||h||) quand ||h|| -> 0

Visualisation: Approximation par la tangente

Interpretation geometrique

La differentielle df(x) est l'application lineaire qui approxime le mieux f au voisinage de x.

C'est la generalisation de la notion de derivee aux espaces de dimension infinie.

II. Proprietes fondamentales

1. Unicite: Si f est differentiable, df(x) est unique
2. Continuite: f differentiable => f continue
3. Linearite: df(x) in L_c(E,F)

III. Exemples

Exemple 1: Application lineaire

Si L: E -> F est lineaire et continue, alors L est differentiable en tout point et:

d_x L(h) = L(h)

Exemple 2: Application bilineaire

f(x,y) = xy sur R^2. On montre que:

d_{(x,y)} f(h,k) = x*k + y*h

Exemple 3: Application constante

Si f est constante, alors f est differentiable et:

df(h) = 0

Visualisation: Differentielle de f(x,y) = xy

Chapitre 3: Derivees Directionnelles

I. Definition

Soient E et F deux e.v.n, U un ouvert de E et v un vecteur non nul de E.

On dit que f: U -> F admet au point x de U une derivee dans la direction de v si:

phi_v(t) = f(x + tv) est derivable en t = 0
D_v f(x) = lim_{t->0} [f(x+tv) - f(x)] / t = phi'_v(0)

Visualisation interactive: Derivees directionnelles

II. Lien avec la differentielle

Proposition fondamentale

Si f est differentiable au sens de Frechet en x, alors pour tout v != 0:

D_v f(x) = d_x f(v)

Surface z = x^2 + y^2 avec directions

Exemple numerique

f(x,y) = x^2 + y^2 au point (1,1):

  • Gradient: nabla f = (2x, 2y) = (2, 2)
  • D_{(1,0)} f = nabla f . (1,0) = 2
  • D_{(0,1)} f = nabla f . (0,1) = 2
  • D_{(1,1)/sqrt(2)} f = nabla f . v = 2*0.707 + 2*0.707 = 2.83 (MAXIMUM)

La direction du gradient donne la croissance maximale!

Chapitre 4: Norme d'Operateur

I. Definition

Soient (E, ||.||_E) et (F, ||.||_F) deux espaces vectoriels normes.

On note L_c(E,F) l'ensemble des applications lineaires continues de E dans F.

||u||_{L_c(E,F)} = sup_{||x||_E <= 1} ||u(x)||_F

Visualisation: Sphere unite -> Ellipsoide

II. Definitions equivalentes

||u|| = sup_{||x|| <= 1} ||u(x)|| (boule unite fermee)
||u|| = sup_{||x|| = 1} ||u(x)|| (sphere unite)
||u|| = sup_{x != 0} ||u(x)|| / ||x|| (rapport de croissance)

Interpretation

La norme d'operateur mesure le "facteur d'etirement" maximal de u.

C'est le plus grand coefficient k tel que: ||u(x)|| <= k * ||x||

Exercice resolu

E = C^0([0,1], R) avec ||f||_inf = sup |f(t)|

u: f -> f(0). Calculer ||u||.

Solution: ||u|| = sup |f(0)|/||f||_inf <= 1. Avec f_0(t) = 1, on a |f_0(0)|/||f_0|| = 1.

Donc ||u|| = 1.

Visualisation: Norme d'operateur en action

III. Propriete fondamentale

Theoreme

Si (F, ||.||_F) est un espace de Banach, alors L_c(E,F) muni de la norme operateur est aussi un espace de Banach.

Preuve en 4 etapes:
1. Convergence simple: u_n(x) -> u(x)
2. u est lineaire
3. u est continue (bornee)
4. u_n -> u dans L_c(E,F)
